# Marcadas_curvas
App para Streamlit. Generadorr de marcadas para corte de tela.

Estaa app permite generar archivos DXF para máquinas de corte de tela. Implementa las siguientes opciones:
- Generación de marcadas en una o varias partes, con selección automática o manual de dirección de corte.
- Generación de marcadas con durva superior.
- Generación de marcadas con recorte específico para sistemas Zip.

Introducción de datos en la aplicación:
1 – Nombre del archivo de marcada
2 – Flecha de tubo
3 – Tipo de marcada
4 – Confección hilo o través
5 – Unidad de medida
6 – Ancho de rollo de tela
7, 8 – Ancho y alto de tela

El botón de Generar marcadas DXF permite obtener uno o varios archivos DXF para introrucir en las máquinas de corte.
